Ex 2.1 — Simulate OVB and verify the formula
Goal: DGP: y = 1 + 2·x + 3·ability + ε, x = 0.7·ability + noise. Show short regression bias equals γ₁ × δ₁ numerically.

# long  <- feols(y ~ x + ability, data.frame(y,x,ability))

# cat("OVB formula:  ", ovb)
